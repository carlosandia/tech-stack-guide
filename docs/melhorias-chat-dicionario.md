# Dicionário de Vícios de Linguagem — Chat & Comunicação em PT-BR

> **Propósito**: Guia de referência para corrigir automaticamente ou orientar usuários sobre vícios de linguagem, abreviações informais e erros ortográficos/gramaticais comuns no português brasileiro usado em chats, WhatsApp e redes sociais.
>
> **Uso no sistema**: Este documento serve como base para validação de texto em campos de mensagem, sugestões de autocorreção e treinamento de modelos de linguagem do CRM.

---

## 2. Erros Ortográficos Comuns

### 2.1 Troca de Letras com mesmo som

| Errado | Correto | Regra |
|--------|---------|-------|
| `obcessão` | obsessão | Prefixo "ob-" + "sessão" |
| `excessão` | exceção | Não existe "excessão" |
| `exeção` | exceção | — |
| `extratégia` | estratégia | Sem "ex" no início |
| `sugestão` / `sujeição` (confundidos) | sugestão / sujeição | Palavras distintas |
| `cheque` (banco) / `xeque` (xadrez) | cheque / xeque | Grafias distintas por contexto |
| `enxergar` | enxergar | X, não CH |
| `encher` | encher | CH, não X |
| `mexer` | mexer | X |

| `expião` | espião | Prefixo errado |
| `estereótipo` | estereótipo | Acento no "ó" |
| `beneficiência` | beneficência | Sem "i" extra |
| `consciência` | consciência | Com "c" |
| `consciente` | consciente | — |
| `exigência` | exigência | Com "x" |

## 3. Erros Gramaticais Comuns

### 3.1 Por que / Porque / Por quê / Porquê

| Forma | Uso Correto | Exemplo |
|-------|-------------|---------|
| `por que` | Em perguntas ou antes de "que" relativo | "Por que você foi?" |
| `porque` | Em respostas / explicações (= pois) | "Fui porque precisei." |
| `por quê` | Em final de frase ou antes de ponto | "Não sei por quê." |
| `o porquê` | Substantivo (= a razão) | "Quero saber o porquê." |
| `pq` *(vício)* | Ambíguo — não use em contextos formais | — |

### 3.2 Mal × Mau / Bem × Bom

| Errado | Correto | Regra |
|--------|---------|-------|
| `Ele é mal caráter` | Ele é mau caráter | "Mau" = adjetivo (oposto de bom) |
| `Está funcionando mau` | Está funcionando mal | "Mal" = advérbio (oposto de bem) |
| `Bem feito` | Bem-feito | Com hífen quando adjetivo composto |
| `Bom dia a todos` | Bom dia a todos | Correto |

### 3.3 Há × A (tempo)

| Errado | Correto | Regra |
|--------|---------|-------|
| `Daqui há cinco minutos` | Daqui a cinco minutos | Futuro → "a" |
| `Aconteceu a dois dias` | Aconteceu há dois dias | Passado → "há" |
| `Faz a 3 anos` | Faz 3 anos / Há 3 anos | — |

### 3.4 Onde × Aonde

| Errado | Correto | Regra |
|--------|---------|-------|
| `Onde você vai?` | Aonde você vai? | "Aonde" = destino (movimento) |
| `Aonde você está?` | Onde você está? | "Onde" = localização (sem movimento) |

### 3.5 Pronomes — Posição e Uso

| Errado | Correto | Observação |
|--------|---------|------------|
| `Me dá` (início de frase formal) | Dá-me | Próclise informal; ênclise = formal |
| `Eu vi ele` | Eu o vi | Pronome oblíquo como objeto direto |
| `Pra mim fazer` | Para eu fazer | "Mim" não sujeito de infinitivo |
| `Entre eu e você` | Entre mim e você | "Mim" após preposição |

### 3.6 Concordância Verbal Comum

| Errado | Correto | Observação |
|--------|---------|------------|
| `Faz dois anos que...` | Faz dois anos que... | Correto — "fazer" impessoal não flexiona |
| `Fazem dois anos` | Faz dois anos | "Fazer" = verbo impessoal |
| `Houveram problemas` | Houve problemas | "Haver" impessoal não flexiona |
| `Haviam muitos` | Havia muitos | Idem |
| `Existe vários` | Existem vários | "Existir" flexiona com sujeito plural |

### 3.7 Crase

| Errado | Correto | Observação |
|--------|---------|------------|
| `Vou a escola` | Vou à escola | Feminino + "a" → crase |
| `Referente a proposta` | Referente à proposta | — |
| `Às vezes` | Às vezes | Crase obrigatória |
| `A vezes` | Às vezes | — |
| `Graças à Deus` | Graças a Deus | Sem crase antes de nome próprio masculino |

### 3.8 Confusão entre Palavras Parecidas

| Par confundido | Forma 1 | Forma 2 | Diferença |
|----------------|---------|---------|-----------|
| `seção / sessão / cessão` | seção (divisão) | sessão (reunião/cinema) | cessão (cedência) |
| `a fim / afim` | a fim de (propósito) | afim (parecido) | — |
| `senão / se não` | senão (caso contrário) | se não (condicional) | — |
| `acerca / há cerca` | acerca de (sobre) | há cerca de (aproximadamente) | — |
| `haja / aja` | haja (haver) | aja (agir) | — |
| `vem / veem` | vem (ele vem) | veem (eles veem) | — |
| `tem / têm` | tem (ele tem) | têm (eles têm) | Acento diferencial |
| `vem / vêm` | vem (ele vem) | vêm (eles vêm) | Acento diferencial |
| `a gente / agente` | a gente (nós) | agente (espião/representante) | — |
| `perda / perca` | perda (substantivo) | perca (verbo) | — |
| `mal / mau` | mal (advérbio) | mau (adjetivo) | — |
